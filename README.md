# 🌌 Vyrex Sora

> A journey that never truly ends.

Vyrex Sora is a modern anime web app where you can explore, track, and discover your favorite anime using a clean UI and smooth experience.

---

## 🚀 Features

- 🎬 Browse trending & top anime
- 📺 View detailed anime info
- ⭐ See ratings, genres, and synopsis
- 👥 Explore characters
- 📌 Add anime to your personal list
- 🗑️ Remove anime from your list
- 📱 Fully responsive design

---

## 🛠️ Built With

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  
- Jikan API (MyAnimeList API)

---

## 🌐 Live Website

👉 [Visit Vyrex Sora](https://vyrexsora.netlify.app)

---

## ⚡ How to Use

1. Browse anime from homepage  
2. Click any anime to view details  
3. Add anime to your list  
4. Manage your list anytime
